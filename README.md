# daily-coding

## daily-coding を使用する

### セットアップ手順

.bashrc に以下を追加します.

    source /PATH-TO-REPOSITORY-DIR/src/main/daily-coding.bashrc

daily-coding コマンドが使えるようになります.

### daily-coding コマンドの使い方

コマンドのヘルプを参照してください.

    daily-coding help

## daily-coding を開発する

### 各種手順

    # テストを実行する.
    make test

    # 作業ファイルを削除する.
    make clean
